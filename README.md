# Portfolio Website

A modern, responsive portfolio website showcasing web development projects and skills. Built with clean HTML, CSS, and vanilla JavaScript, featuring smooth animations, a collapsible sidebar navigation, and an elegant user interface.

## Features

- **Responsive Design**: Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices
- **Interactive Sidebar Navigation**: Collapsible sidebar menu with smooth transitions and hover effects
- **Scroll Animations**: Intersection Observer API for smooth fade-in animations as elements appear on scroll
- **Parallax Effects**: Dynamic parallax scrolling on the header for visual depth
- **Modern UI/UX**: Clean, professional design with gradient backgrounds, card-based layouts, and smooth transitions
- **Project Showcase**: Grid-based project display with hover effects and external GitHub links
- **Smooth Scrolling**: Anchor link navigation with smooth scroll behavior

## Tech Stack

- **HTML5**: Semantic markup with modern best practices
- **CSS3**: Custom properties (CSS variables), flexbox, animations, and responsive design
- **JavaScript (ES6+)**: Vanilla JavaScript for interactivity, Intersection Observer API for scroll animations

## Sections

1. **Home/Header**: Eye-catching hero section with gradient background and animated content

## Key Technical Highlights

- **CSS Custom Properties**: Consistent theming with CSS variables
- **Advanced Animations**: Multiple keyframe animations and transition effects
- **Intersection Observer**: Performant scroll-based animations
- **Event Delegation**: Efficient event handling for sidebar and navigation
- **Smooth Scroll Behavior**: Enhanced user experience with smooth anchor navigation
- **Accessibility**: Semantic HTML and keyboard-friendly navigation
